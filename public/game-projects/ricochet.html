This project was made as a rapid prototype during the Spring 2026 semester for Game Design II class at Bradley. My team challenged ourselves to try using Unreal Engine for the first time for this prototype to learn a new engine.

Because I mainly focus on programming, I was responsible for handling technical setup and management of the project, helping my team solve issues with the new engine. Part of that was learning Blueprint coding so I could help teach it to my teammates, and learning Unreal's various settings.

My Contributions:

  • Used Unreal's Behaviour Tree system to design enemy AI.
  • Programmed main player actions and checkpoints using Bleprints.
  • Enforced GitHub pull request reviews prevent merge conflicts and corruptions.
  • Helped a teammate implement UI for the HUD and pause menu.
